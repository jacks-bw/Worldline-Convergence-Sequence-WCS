## 摘要
本文提出一种基于**全域字符计频与首次出现顺序拼接**规则的字符迭代序列，以《命运石之门》中的“世界线收束”理论为类比框架，对序列的演化过程进行系统化定义与研究。命名为——世界线收束序列（Worldline Convergence Sequence），后文简称WCS序列）。该序列支持数字、字母、特殊符号等任意字符作为初始种子，通过固定规则逐代推演。本文通过鸽巢原理严格证明：**任意有限长度的初始种子，经过有限次迭代后必然进入循环状态，即完成“世界线收束”**。在此基础上，本文设计并实现了序列生成与闭环检测算法，可自动识别序列的“分歧世界域”与“命运收束闭环”，并提供交互式推演程序。该研究既是对经典外观数列的创意性扩展，也为字符迭代系统的收敛性分析提供了直观的案例模型。

**关键词**：字符迭代序列；外观数列变种；循环检测；收敛性证明；世界线收束模型

## 一、引言
### 1.1 研究背景
外观数列（Look-and-say sequence）是经典的自描述迭代序列，其核心规则为“从左到右统计连续相同字符的数量，并以‘数量+字符’的形式生成下一项”。该序列具有丰富的数论性质，已被证明存在生长常数、渐近模式等特征。

本文在经典外观数列的基础上，提出一种**全域计频变体规则**：不再统计连续字符，而是统计字符串中所有字符的全局出现频次，并按照字符首次出现的顺序拼接生成下一项。该规则打破了“连续分组”的限制，支持任意Unicode字符作为输入，极大扩展了序列的适用范围与演化形态。

### 1.2 世界线收束理论的概念映射
为直观描述序列的演化特征，本文引入科幻作品中的“世界线收束”理论作为类比框架，将抽象的迭代过程映射为世界线的演化与收束，核心术语对应关系如下：

| 数学概念 | 世界线收束对应术语 | 定义说明 |
|---------|-------------------|---------|
| 初始字符串 | 世界线原点种子 | 迭代的起始输入，可由任意字符构成 |
| 单次迭代生成新字符串 | 世界线跃迁 | 遵循固定规则生成下一代节点的过程 |
| 迭代项 | 世界线节点 | 每一次迭代生成的字符串，对应一条世界线的状态 |
| 前缀非循环段 | 分歧世界域 | 进入循环前的演化路径，代表世界线的分歧阶段 |
| 循环周期段 | 命运收束闭环 | 重复出现的节点序列，代表世界线收束后的固定宿命 |
| 循环周期长度 | 闭环阶数 | 一个完整循环包含的节点数量 |
| 序列进入循环 | 世界线收束 | 序列从自由演化进入固定循环的状态转变 |

该映射并非物理意义上的理论，而是一种创意化的研究叙事，旨在为字符迭代系统的收敛性分析提供更具辨识度的表述体系。

## 二、核心迭代规则与形式化定义
### 2.1 单步迭代规则
对于任意一个非空字符串 $S$，其下一代节点 $S'$ 的生成严格遵循三步规则：

1. **全域字符计频**
   遍历字符串 $S$ 的每一个字符，统计每个字符的总出现次数，得到频次映射 $f(c)$，其中 $c$ 为字符，$f(c)$ 为该字符在 $S$ 中出现的总次数。该统计不区分字符是否连续，仅计算全局总频次。

2. **首次出现顺序去重**
   再次从左到右遍历字符串 $S$，记录每个字符第一次出现的位置顺序，去除重复字符，得到有序字符列表 $O = [c_1, c_2, ..., c_k]$，其中 $k$ 为字符串中不同字符的种类数。该列表保留字符在原字符串中的原生先后顺序。

3. **顺序拼接生成新节点**
   按照有序列表 $O$ 的顺序，将每个字符拼接为“频次的十进制字符串 + 字符本身”的形式，依次连接后得到下一代节点 $S'$，即： $S' = \sum_{i=1}^{k} \left( \text{str}(f(c_i)) + c_i \right)$

### 2.2 序列定义
给定初始种子 $S_0$，世界线收束序列定义为通过上述规则逐代迭代生成的无穷序列 $\{S_0, S_1, S_2, ...\}$，其中 $S_{n+1} = F(S_n)$，$F$ 为上述单步迭代函数。

## 三、收束性定理与结构性质
### 3.1 字符集封闭性
**性质1（字符集封闭）**：序列演化过程中出现的所有字符，均属于“初始种子字符集 ∪ 阿拉伯数字0-9”的有限集合。
- 证明：每一代新节点仅由上一代字符的频次（十进制数字）和上一代字符本身拼接而成，不会引入初始种子与数字之外的新字符。因此整个序列的字符全集始终是有限集。

### 3.2 序列长度的有界性
**性质2（长度有上界）**：序列的字符串长度不会无限增长，存在确定的上界。
- 证明：设当前字符串长度为 $L$，包含 $k$ 种不同字符，则下一代字符串的长度为各频次的位数之和加 $k$。由于单个字符的最大出现次数为 $L$，其十进制位数为 $O(\log_{10} L)$，因此下一代总长度满足：$L' \leq k \cdot (\lfloor \log_{10} L \rfloor + 1) + k$
	当 $L$ 增长到一定阈值后，必有 $L' < L$，即长度开始下降。因此序列长度不可能无限递增，存在全局上界。

### 3.3 必然收束定理
**定理（世界线必然收束定理｜专属WCS序列定理）**：任意有限初始种子生成的序列，经过有限次迭代后必然进入循环状态，即完成收束。
- 证明：由性质1与性质2可知，序列的字符集有限、字符串长度有上界，因此所有可能出现的字符串总数是有限的。根据鸽巢原理，在无穷迭代过程中必然会出现重复的节点。一旦某个节点第二次出现，后续迭代将完全重复之前的路径，序列即进入循环，完成“收束”。

### 3.4 收束结构的标准形式
任意序列的完整演化路径均可拆分为两部分：
1. **分歧世界域（前缀段）**：从初始种子到首次进入循环的前一个节点为止，这部分节点仅出现一次，代表世界线的自由分歧阶段。
2. **命运收束闭环（循环段）**：重复出现的节点序列，这部分节点会无限循环，代表世界线收束后的固定宿命。闭环包含的节点数称为**闭环周期长度**。

特殊地，当闭环周期长度为1时，该节点称为**不动点**，即迭代后结果与自身完全相同，代表世界线收束于单一稳定状态。

## 四、算法设计与工程实现
### 4.1 世界线节点生成算法
算法支持两种推演模式：
- **指定行数模式**：按照用户指定的迭代次数生成对应数量的节点，适用于观察局部演化过程。
- **自动收束模式**：持续迭代直到出现重复节点，返回完整的演化路径，适用于观测完整收束过程。

原算法采用列表存储历史节点，为直观展示闭环闭合，检测到重复节点时仍会将重复节点追加至结果末尾，使用户可直接观察到“首尾衔接”的闭环特征。优化版本采用字典存储节点与索引的映射，将查重时间复杂度从 $O(n)$ 降至 $O(1)$，大幅提升长序列推演效率。

### 4.2 闭环结构检测算法
采用 Floyd 判圈思想的变体实现，通过边迭代边记录索引的方式，可一次性定位循环起点与周期长度：
1. 遍历序列，用字典记录每个节点第一次出现的索引位置；
2. 当遇到已记录的节点时，其第一次出现的位置即为循环起点；
3. 从循环起点到当前位置之前的节点序列，即为完整的闭环，二者索引差即为周期长度。

该算法可自动输出分歧域节点列表、闭环节点列表与周期长度，完成结构化的收束研判。

### 4.3 交互式系统设计
系统提供命令行交互入口，具备以下特性：
- 全字符兼容：支持数字、大小写字母、特殊符号、混合字符等任意输入；
- 双模式切换：输入数字指定推演行数，空回车自动推演至收束完成；
- 结构化输出：同时输出逐节点推演日志与收束研判报告，兼顾过程展示与结论呈现。
