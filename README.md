# 世界线收束序列（Worldline Convergence Sequence, WCS）

基于 **全域字符计频 + 首次出现顺序拼接** 的自研迭代规则，结合《命运石之门》世界线收束理论构建的字符序列收敛模型。数学上已证明：**任意有限初始种子，经有限步迭代后必然进入循环——即"世界线收束"**。

## 快速开始

```bash
# 交互式推演（输入种子，回车自动收束）
python wcs.py

# 批量校验并生成分析报告（支持范围 1~999）
echo 1~999 | python wcs_verify.py
```

### wcs.py — 交互式推演

```
输入世界线原点种子: 1

节点  1 (分歧域): 1
节点  2 (分歧域): 11
节点  3 (分歧域): 21
  ...
节点 14 (闭环起点): 23322114
节点 15 (闭环节点): 32232114

========== 收束研判报告 ==========
分歧域长度: 13 节点
宿命闭环周期: 2
宿命闭环: [23322114, 32232114]
```

### wcs_verify.py — 批量校验 & 报告生成

输入种子范围（如 `1~999`），自动批量推演并生成包含散点图、折线图、饼图、汇总数据表的 HTML 专业分析报告。

## 核心原理

1. **全域计频**：统计字符串中每个字符的总出现次数
2. **首次顺序去重**：按字符首次出现顺序排列
3. **拼接生成**：将每个字符拼接为 `"频次+字符"` 的形式
